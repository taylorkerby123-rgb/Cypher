# Kerby Cipher

A personal symbol cipher encoder/decoder — optimized for mobile.

## Deploy to GitHub Pages (free, 5 minutes)

### Step 1 — Create a GitHub account
Go to [github.com](https://github.com) and sign up (free).

### Step 2 — Create a new repository
1. Click the **+** icon → **New repository**
2. Name it `kerby-cipher` (or anything you like)
3. Set to **Public**
4. Click **Create repository**

### Step 3 — Upload the files
1. On your new repo page, click **uploading an existing file**
2. Drag all files from this ZIP into the upload area:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon.png`
   - `.nojekyll`
3. Click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to **Settings** → **Pages** (left sidebar)
2. Under **Source**, select **Deploy from a branch**
3. Choose **main** branch, **/ (root)** folder
4. Click **Save**

### Step 5 — Get your URL
After ~1 minute, your app is live at:
```
https://YOUR-USERNAME.github.io/kerby-cipher/
```

Bookmark it on your iPhone — tap **Share → Add to Home Screen** to install it as an app icon.
